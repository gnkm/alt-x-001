# Product Mission

## Pitch

**Alt X** は、個人向けの文章投稿ウェブアプリです。
自分だけのプライベートな空間で、思考やアイデアを自由に記録・整理したい個人ユーザーを支援します。
シンプルな操作性と効率的な検索機能により、蓄積した投稿を価値ある情報資産として活用できます。

## Users

### Primary Customers

- **個人ユーザー**: 自分専用の文章投稿・管理を行いたい方

### User Personas

**個人利用者** (20-50代)
- **Role:** 日常的にメモや考えを記録したい個人
- **Context:** プライベートな環境で、SNSに公開せずに自分だけの投稿を管理したい
- **Pain Points:**
  - 既存のSNSは他人の目が気になり、自由に書けない
  - メモアプリは検索性が低く、過去の記録を見つけにくい
  - 画像付きの投稿を整理して保存する場所がない
- **Goals:**
  - 気軽に思考やアイデアを記録できる
  - 過去の投稿を素早く検索・参照できる
  - タグで効率的に分類・整理できる

## The Problem

### 自分だけの投稿を効率的に管理できる場所がない

既存のSNSは公開前提であり、プライベートなメモや思考を気軽に残せません。
一方、メモアプリは検索性や分類機能が不十分で、蓄積した情報を活用しにくいという問題があります。

**Our Solution:** ログイン認証による完全プライベートな環境と、タグ付け・検索機能を備えた文章投稿システムを提供します。

## Differentiators

### シンプルさと実用性の両立

一般的なSNSと異なり、Alt Xは完全に自分専用の空間として設計されています。
余計な機能を排除し、投稿・検索・整理という核心機能に集中することで、迷いなく使い続けられます。

### ローカル環境での完全なプライバシー

クラウドサービスと異なり、自分のローカル環境で動作するため、データは完全に自分の管理下に置かれます。
これにより、プライバシーを重視するユーザーに安心感を提供します。

## Key Features

### Core Features

- **投稿機能:** テキストと画像を組み合わせて、思考やアイデアを自由に記録できます
- **タグ付け:** 投稿にタグを付けることで、後から関連する投稿をまとめて参照できます
- **投稿削除:** 不要になった投稿を簡単に削除できます

### 検索・閲覧機能

- **投稿一覧表示:** 遅延ロードにより、大量の投稿もスムーズに閲覧できます
- **投稿検索:** キーワードやタグで過去の投稿を素早く見つけられます

### セキュリティ機能

- **ログイン機能:** 認証により、自分だけがアクセスできる安全な環境を確保します
